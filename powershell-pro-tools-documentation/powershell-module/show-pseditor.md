# Show-PSEditor

`Show-PSEditor` displays a terminal-based PowerShell editor that contains the following features.&#x20;

* IntelliSense
* Syntax Highlighting&#x20;
* Execution
* Syntax Error Tagging&#x20;
* File Save and Open&#x20;
